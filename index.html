GET/leagues

Method: GET Route: https://placeholder.url/leagues Headers: Authorization: Bearer YOUR_API_KEY_HERE Description: Returns a list of all Pokémon Go Battle leagues.

GET/leagues/great-league/leads

Method: GET Route: https://placeholder.url/leagues/great-league/leads Headers: Authorization: Bearer YOUR_API_KEY_HERE Description: Returns a list of all lead Pokémon players have encountered in Great League.

GET/leagues/great-league/leads/{slug}

Method: GET Route: https://placeholder.url/leagues/great-league/leads Headers: Authorization: Bearer YOUR_API_KEY_HERE Description: Returns the number of times the Pokémon with the associated slug has been encountered in Great League, and the ratings the encounters happened.

POST/leagues/great-league/leads

Method: POST Route: https://placeholder.url/leagues/great-league/leads Headers: Content-Type: application/json Authorization: Bearer YOUR_API_KEY_HERE Body: { "name": "Charizard", "slug": "charizard", "rating": 1500 } Description: Creates a new Great League lead entry from the data provided in the body.

DELETE/leagues/great-league/leads/{slug}

Method: DELETE Route: https://placeholder.url/leagues/great-league/leads/{slug} Headers: Authorization: Bearer YOUR_API_KEY_HERE Description: Deletes the Pokémon Great League lead entry associated with the provided slug.
